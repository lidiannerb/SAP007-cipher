Digite a chave para Cifrar ou Decifrar suas mensagens

Orientações de uso

Digite a chave (número) no campo "offset"

Para CIFRAR

Digite a mensagem dentro do bloco 1, aperte no botão "CIFRAR" e obtenha a mensagem cifrada dentro do bloco 2

Para DECIFRAR

Digite a mensagem cifrada dentro do bloco 3, aperte no botão "DECIFRAR" e obtenha a mensagem no bloco 2

Esta aplicação não aceita 'ç' ou caracteres especiais/p>
